## 测试执行流程

### 1. 环境准备
```bash
# 设置测试环境
./e2e-tests/run-e2e-tests.sh --setup-only

# 或清理环境
./e2e-tests/run-e2e-tests.sh --clean
```

### 2. 执行测试
```bash
# 运行所有测试
./e2e-tests/run-e2e-tests.sh

# 运行特定浏览器测试
./e2e-tests/run-e2e-tests.sh -b firefox

# 运行特定测试文件
./e2e-tests/run-e2e-tests.sh -t tests/auth/login.spec.ts

# 并行运行测试
./e2e-tests/run-e2e-tests.sh -p -b chromium
```

### 3. 查看报告
```bash
# 生成并查看报告
./e2e-tests/run-e2e-tests.sh --no-report
npx allure open e2e-tests/reports/allure-report
```

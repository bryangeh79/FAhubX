## 🏗️ 项目架构

### 模块化设计
```
src/
├── vpn/                    # VPN核心模块
│   ├── openvpn/           # OpenVPN实现
│   ├── wireguard/         # WireGuard实现  
│   ├── ip-manager/        # IP管理
│   └── network-isolation/ # 网络隔离
├── shared/                # 共享模块
│   ├── types/            # TypeScript类型
│   ├── utils/            # 工具函数
│   └── config/           # 配置管理
└── api/                  # API接口层
    └── vpn/              # VPN管理API
```

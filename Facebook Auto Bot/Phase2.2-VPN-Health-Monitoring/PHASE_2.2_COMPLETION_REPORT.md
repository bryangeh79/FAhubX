## API接口

### VPN管理接口 (`/api/vpn`)
- `GET /status` - 获取VPN状态
- `POST /openvpn/connect` - 连接OpenVPN
- `POST /wireguard/connect` - 连接WireGuard
- `POST /:id/disconnect` - 断开VPN连接
- `POST /rotate-ip` - 轮换IP地址
- `GET /test/performance` - 测试网络性能

### 健康监控接口 (`/api/health`)
- `POST /start` - 启动健康监控
- `POST /stop` - 停止健康监控
- `POST /run` - 立即运行健康检查
- `GET /status` - 获取健康状态
- `GET /history` - 获取检查历史
- `PUT /config` - 更新监控配置

### 监控告警接口 (`/api/monitoring`)
- `GET /status` - 获取监控状态
- `GET /metrics` - 获取性能指标
- `GET /alerts` - 获取告警列表
- `POST /alerts/:id/acknowledge` - 确认告警
- `GET /dashboard` - 获取仪表板数据
- `GET /updates` - 实时更新流

## 部署配置

### 环境要求
- Node.js 18+
- Docker & Docker Compose
- PostgreSQL 15+
- Redis 7+
- Linux内核支持网络命名空间

### 快速部署
```bash
# 1. 克隆项目
git clone <repository-url>

# 2. 配置环境变量
cp .env.example .env
# 编辑.env文件配置数据库、密钥等

# 3. 启动服务
```

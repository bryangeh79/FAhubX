## 打包脚本

### Linux/macOS
```bash
#!/bin/bash

# 创建临时目录
mkdir -p /tmp/fbautobot-blueprint

# 复制所有文档到临时目录
cp README.md /tmp/fbautobot-blueprint/
cp 1.项目概述.md /tmp/fbautobot-blueprint/
cp 2.系统架构总览.md /tmp/fbautobot-blueprint/
cp 3.技术栈选择.md /tmp/fbautobot-blueprint/
cp 4.核心模块设计-简洁版.md /tmp/fbautobot-blueprint/
cp 5.数据库设计.md /tmp/fbautobot-blueprint/
cp 6.API接口规范-简洁版.md /tmp/fbautobot-blueprint/
cp 7.前端架构-简洁版.md /tmp/fbautobot-blueprint/
cp 8.部署架构-简洁版.md /tmp/fbautobot-blueprint/
cp 9.实施路线图.md /tmp/fbautobot-blueprint/
cp 10.风险管理.md /tmp/fbautobot-blueprint/
cp 11.交付物清单.md /tmp/fbautobot-blueprint/

# 可选：复制完整版文档
# cp 4.核心模块设计.md /tmp/fbautobot-blueprint/
# cp 6.API接口规范.md /tmp/fbautobot-blueprint/
# cp 7.前端架构.md /tmp/fbautobot-blueprint/
# cp 8.部署架构.md /tmp/fbautobot-blueprint/

# 创建压缩包
cd /tmp
tar -czf fbautobot-engineering-blueprint-v1.0.tar.gz fbautobot-blueprint/

# 移动回当前目录
mv fbautobot-engineering-blueprint-v1.0.tar.gz /workspace/Facebook\ Auto\ Bot/

echo "打包完成！文件位置: /workspace/Facebook Auto Bot/fbautobot-engineering-blueprint-v1.0.tar.gz"
```

## 手动打包步骤

如果脚本无法运行，可以手动打包：

1. **创建新文件夹**：`fbautobot-engineering-blueprint-v1.0`
2. **复制所有文档**：将上述12个文档复制到文件夹中
3. **压缩文件夹**：使用压缩工具（如7-Zip、WinRAR）压缩文件夹
4. **命名压缩包**：`fbautobot-engineering-blueprint-v1.0.zip`

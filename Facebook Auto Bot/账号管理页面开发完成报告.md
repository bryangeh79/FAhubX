## 技术实现

### 技术栈
- **前端框架**: React 18 + TypeScript
- **UI组件库**: Ant Design 5
- **表单处理**: React Hook Form 7.45 + Zod 3.21
- **状态管理**: React Query 3.39
- **构建工具**: Vite 4.4
- **图表库**: Recharts 2.7
- **路由**: React Router DOM 6.14

## 功能特性

### 账号管理功能
1. **账号列表**: 显示所有Facebook账号，支持分页、搜索、筛选
2. **状态显示**: 活跃、禁用、封禁、暂停状态标签
3. **统计信息**: 任务总数、成功率、最后活动时间
4. **操作支持**: 查看详情、编辑、测试连接、手动登录、删除

### 批量操作功能
1. **批量选择**: 支持多选和全选
2. **批量操作**: 启动、暂停、删除、测试、导出
3. **进度跟踪**: 实时显示操作进度
4. **历史记录**: 保存最近的操作记录

### VPN配置功能
1. **配置管理**: 添加、编辑、删除VPN配置
2. **连接测试**: 测试VPN连接状态和性能
3. **轮换策略**: 随机、轮询、最少使用、固定策略
4. **性能监控**: 延迟、带宽等性能指标

### 表单功能
1. **验证机制**: 客户端表单验证
2. **2FA支持**: 双重验证配置
3. **批量导入**: 支持CSV文件导入
4. **VPN集成**: 与VPN配置无缝集成

## 部署说明

### 环境要求
- Node.js >= 16.0.0
- npm >= 8.0.0
- 现代浏览器（Chrome 90+, Firefox 88+, Safari 14+）

### 构建步骤
```bash
# 安装依赖
cd frontend
npm install

# 开发模式运行
npm run dev

# 构建生产版本
npm run build

# 类型检查
npm run type-check

# 运行测试
npm test
```

### 配置说明
1. 复制 `.env.example` 为 `.env`
2. 配置 API 基础URL
3. 配置其他环境变量

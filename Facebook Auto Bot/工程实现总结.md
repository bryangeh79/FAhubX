## 技术架构实现

### 后端架构
- **框架**: NestJS + TypeScript
- **数据库**: PostgreSQL 16 + TypeORM
- **缓存**: Redis 7
- **消息队列**: RabbitMQ
- **对象存储**: MinIO (S3兼容)
- **认证**: JWT + 会话管理
- **API文档**: Swagger/OpenAPI

### 前端架构
- **框架**: React 19 + TypeScript
- **构建工具**: Vite
- **UI组件**: Ant Design 5
- **状态管理**: Zustand
- **路由**: React Router 6
- **HTTP客户端**: Axios
- **PWA支持**: 已配置

### 开发运维
- **容器化**: Docker + Docker Compose
- **编排**: Kubernetes (生产环境)
- **CI/CD**: GitHub Actions
- **监控**: Prometheus + Grafana + Loki
- **日志**: Winston + 结构化日志

## 快速开始指南

### 1. 环境准备
```bash
# 安装 Docker 和 Docker Compose
# 克隆项目代码
git clone <repository-url>
cd facebook-auto-bot
```

### 2. 启动开发环境
```bash
# 启动所有服务
./start-dev.sh up

# 或分步启动
./start-dev.sh install    # 安装依赖
./start-dev.sh db migrate # 数据库迁移
./start-dev.sh db seed    # 种子数据
```

### 3. 访问应用
- 前端应用: http://localhost:8080
- 后端API: http://localhost:3000
- API文档: http://localhost:3000/api-docs
- 数据库: localhost:5432 (postgres/password)
- Redis: localhost:6379
- RabbitMQ管理: http://localhost:15672 (admin/password)
- MinIO控制台: http://localhost:9001 (minioadmin/minioadmin)

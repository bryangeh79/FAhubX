## 第一章：系统介绍

### 1.1 系统概述
Facebook Auto Bot是一个多租户SaaS平台，专门为企业和个人提供Facebook页面自动化管理服务。系统支持同时管理多个Facebook账号，实现自动化内容发布、智能消息回复、数据分析和团队协作等功能。

### 1.2 核心功能特性
- ✅ **多账号管理**：支持同时管理最多10个Facebook账号
- ✅ **VPN/IP集成**：支持通过不同VPN/IP登录Facebook账号
- ✅ **自动化任务**：支持定时发布、自动回复、智能对话
- ✅ **实时监控**：Dashboard实时查看所有账号状态
- ✅ **PWA支持**：支持渐进式Web应用，可在移动设备上使用
- ✅ **团队协作**：支持多用户协作管理账号

### 1.3 系统架构
系统采用前后端分离架构：
- **前端**：React + TypeScript + Ant Design
- **后端**：Node.js + NestJS + TypeScript
- **数据库**：PostgreSQL + Redis
- **部署**：Docker + Kubernetes

### 1.4 浏览器兼容性
- Chrome 90+ ✅
- Firefox 88+ ✅
- Safari 14+ ✅
- Edge 90+ ✅
- 移动端浏览器 ✅

## 第二章：快速开始

### 2.1 注册账号
1. 访问系统登录页面
2. 点击"注册新账号"按钮
3. 填写注册信息：
   - 邮箱地址
   - 密码（至少8位，包含字母和数字）
   - 确认密码

4. 点击"注册"按钮
5. 查收验证邮件并完成验证

### 2.2 首次登录
1. 访问系统登录页面
2. 输入注册时使用的邮箱和密码

## 第三章：账号管理

### 3.1 添加Facebook账号
#### 3.1.1 单个账号添加
1. 进入"账号管理"页面
2. 点击"添加账号"按钮
3. 选择登录方式：
   - **方式一：普通网页登录**
     - 输入Facebook账号和密码
     - 完成双重验证（如启用）
     - 授权应用权限
   
   - **方式二：VPN/IP登录**
     - 选择VPN配置或输入IP地址
     - 输入Facebook账号和密码
     - 完成登录验证

4. 配置账号信息：
   - 账号名称（自定义）
   - 账号标签（用于分类）
   - 账号分组（可选）
   - 备注信息

#### 3.1.2 批量导入账号
1. 进入"账号管理"页面
2. 点击"批量导入"按钮
3. 下载导入模板文件
4. 按照模板格式填写账号信息
5. 上传填写好的模板文件
6. 系统自动验证并导入账号

### 3.2 账号状态管理
#### 3.2.1 查看账号状态
账号状态分为以下几种：
- 🟢 **活跃**：账号正常运行中
- 🟡 **空闲**：账号已登录但未执行任务
- 🔴 **错误**：账号出现异常
- ⚫ **禁用**：账号被手动禁用
- 🚫 **封禁**：账号被Facebook封禁

#### 3.2.2 账号操作
对每个账号可以执行以下操作：
- **启动**：激活账号开始执行任务
- **暂停**：临时停止账号任务执行
- **停止**：完全停止账号
- **测试**：测试账号连接状态
- **编辑**：修改账号配置
- **删除**：从系统中移除账号

### 3.3 账号分组管理
#### 3.3.1 创建分组
1. 进入"账号分组"页面
2. 点击"新建分组"按钮
3. 填写分组信息：
   - 分组名称
   - 分组描述
   - 分组颜色标签
4. 选择要加入分组的账号
5. 点击"保存"按钮

#### 3.3.2 分组操作
- **按组执行任务**：对分组内所有账号执行相同任务
- **分组统计**：查看分组整体性能指标
- **分组设置**：配置分组级别的权限和策略

### 3.4 账号健康监控
系统自动监控每个账号的健康状态：

#### 3.4.1 健康指标
- **登录状态**：账号是否保持登录
- **会话有效期**：登录会话剩余时间
- **任务成功率**：最近任务执行成功率
- **响应时间**：平均API响应时间
- **资源使用**：CPU和内存使用情况

#### 3.4.2 健康告警
当账号出现以下问题时，系统会发送告警：
- ❌ 登录失败超过3次
- ⚠️ 任务失败率超过20%
- 🐌 响应时间超过5秒
- 💾 资源使用率超过80%

### 3.5 VPN/IP配置管理
#### 3.5.1 添加VPN配置
1. 进入"VPN/IP管理"页面
2. 点击"添加VPN配置"按钮
3. 填写VPN信息：
   - VPN名称
   - 服务器地址
   - 端口号
   - 协议类型（OpenVPN、WireGuard等）
   - 认证信息
4. 测试VPN连接
5. 保存配置

#### 3.5.2 IP地址池管理
1. 创建IP地址池
2. 添加IP地址到池中
3. 配置IP轮换策略：
   - 按时间轮换
   - 按任务轮换
   - 按账号轮换
4. 设置IP质量监控

## 第四章：任务调度

### 4.1 创建自动化任务
#### 4.1.1 任务类型
系统支持以下类型的自动化任务：

1. **内容发布任务**

   - 发布轮播图

2. **消息回复任务**

3. **互动任务**
   - 自动点赞
   - 自动分享
   - 自动关注
   - 自动加入群组

4. **数据收集任务**
   - 收集帖子数据
   - 收集用户信息
   - 收集竞品数据
   - 收集市场趋势

#### 4.1.2 创建任务步骤
1. 进入"任务调度"页面
2. 点击"新建任务"按钮
3. 选择任务类型
4. 配置任务基本信息：
   - 任务名称
   - 任务描述
   - 执行账号（单个或多个）

5. 配置任务内容：

   - 支持变量替换（如{用户名}、{时间}等）

   - 添加对话步骤

   - 配置回复内容
   - 设置跳转逻辑

5. 配置触发条件：
   - 关键词触发

6. 测试剧本：
   - 使用测试工具模拟对话

#### 5.2.2 高级功能

## 第八章：常见问题

### 8.1 账号相关问题
#### Q1: 如何解决Facebook账号登录失败？
**A:** 可能的原因和解决方案：
1. **账号密码错误**：确认账号密码正确
2. **双重验证**：检查手机验证码或备用代码
3. **账号被限制**：登录Facebook网页版检查账号状态
4. **网络问题**：尝试切换网络或使用VPN
5. **系统维护**：检查系统状态页面

#### Q2: 如何提高账号安全性？
**A:** 建议的安全措施：
1. 启用双重验证
2. 定期更换密码
3. 使用专用VPN/IP
4. 监控异常登录
5. 及时处理安全告警

### 8.2 任务执行问题
#### Q3: 任务执行失败怎么办？
**A:** 排查步骤：

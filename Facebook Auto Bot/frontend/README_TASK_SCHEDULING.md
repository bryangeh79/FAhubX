# 任务调度界面使用指南

## 组件概述

本项目已成功开发了完整的 Facebook Auto Bot 任务调度界面，包含以下核心组件：

### 1. 任务列表页面 (`TasksPage.tsx`)
- 主任务管理界面
- 支持批量操作（选择、启动、暂停、停止、删除）
- 导入/导出功能
- 任务模板保存
- 链接到监控页面

### 2. 任务创建组件 (`TaskCreateForm.tsx`)
- 四步创建流程：
  1. 基本信息
  2. 内容配置
  3. 时间调度
  4. 高级配置

### 3. 时间调度选择器 (`ScheduleSelector.tsx`)
- 支持四种调度方式：
  - 立即执行
  - 定时执行
  - 重复执行
  - Cron表达式

### 4. 对话剧本选择器 (`ScriptSelector.tsx`)
- 浏览和选择预定义剧本
- 按类别筛选
- 剧本预览
- 收藏功能

### 5. 任务详情组件 (`TaskDetail.tsx`)
- 完整任务信息展示
- 执行历史记录
- 任务控制功能

### 6. 任务监控组件 (`TaskMonitor.tsx`)
- 实时监控仪表板
- 性能指标图表
- 系统健康状态

## 页面路由

### 已添加的路由
- `/tasks` - 任务列表页面
- `/tasks/create` - 创建新任务
- `/tasks/:id` - 任务详情页面
- `/tasks/:id/edit` - 编辑任务
- `/tasks/monitor` - 任务监控页面

## API 集成

### 使用的 API 端点
- `GET /tasks` - 获取任务列表
- `POST /tasks` - 创建任务
- `GET /tasks/:id` - 获取任务详情
- `PUT /tasks/:id` - 更新任务
- `DELETE /tasks/:id` - 删除任务
- `POST /tasks/batch` - 批量操作
- `POST /tasks/export` - 导出任务
- `POST /tasks/import` - 导入任务
- `POST /tasks/templates` - 保存模板

## 使用方法

### 1. 创建新任务
```typescript
// 导航到创建页面
navigate('/tasks/create');

// 或者通过任务列表页面的"创建任务"按钮
```

### 2. 批量操作任务
1. 在任务列表页面勾选多个任务
2. 点击"批量操作"下拉菜单
3. 选择要执行的操作（启动、暂停、停止、删除）

### 3. 导入/导出任务
- **导出**: 选择任务后点击"导出"按钮
- **导入**: 点击"导入"按钮，选择JSON文件

### 4. 监控任务执行
- 点击"监控"按钮进入监控页面
- 查看实时执行状态和性能指标

## 注意事项

### 1. 时间调度
- Cron表达式使用国际标准格式
- 时间选择考虑了时区问题
- 重复执行支持复杂的调度规则

### 2. 账号选择
- 最多可选择10个Facebook账号
- 账号状态会实时显示
- 支持并发控制配置

### 3. 剧本选择
- 预定义剧本按类别组织
- 支持剧本预览
- 可以收藏常用剧本

### 4. 错误处理
- 表单验证实时反馈
- API错误友好提示
- 加载状态显示

## 项目技术架构

### 前端架构
- **框架**: React 18 + TypeScript
- **构建工具**: Vite 4.4
- **UI组件**: Ant Design 5
- **状态管理**: Zustand + React Query
- **PWA支持**: Vite PWA插件 + Workbox
- **实时通信**: WebSocket + Socket.io客户端

### 后端架构
- **框架**: NestJS + TypeScript
- **数据库**: PostgreSQL + Redis
- **消息队列**: Bull (基于Redis)
- **实时通信**: WebSocket/Socket.io服务器
- **任务调度**: @nestjs/schedule + node-cron
- **API文档**: Swagger/OpenAPI 3.0

### 自动化架构
- **浏览器自动化**: Puppeteer + Stealth插件
- **人类行为模拟**: 随机延迟 + 自然操作模拟
- **VPN集成**: OpenVPN + WireGuard客户端
- **IP管理**: 智能IP分配和轮换系统
- **会话管理**: AES-256-GCM加密会话存储

### 对话引擎
- **剧本系统**: 50个专业对话剧本
- **NLP引擎**: 意图识别 + 实体提取 + 上下文管理
- **执行器**: Facebook Messenger API集成
- **分析系统**: 对话质量评估 + 优化建议

### 部署架构
- **容器化**: Docker + Docker Compose
- **编排**: Kubernetes配置 (可选)
- **监控**: Prometheus + Grafana + Loki
- **CI/CD**: GitHub Actions自动化流水线
- **备份恢复**: 完整的备份和灾难恢复系统

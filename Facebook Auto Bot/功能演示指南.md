# Facebook Auto Bot 功能演示指南

## 🎯 系统概述

Facebook Auto Bot是一个完整的Facebook账号自动化管理系统，支持10个账号的并行管理，通过Puppeteer进行普通网页登录，具备完整的反检测和监控功能。

## 📊 系统架构

```
前端管理界面 (React + TypeScript + Ant Design)
    ↓
API服务层 (RESTful API)
    ↓
后端服务 (Node.js + Express)
    ↓
Puppeteer浏览器实例 (10个独立实例)
    ↓
Facebook网站 (普通网页登录)
```

## 🚀 核心功能演示

### 1. 账号管理 (AccountsPage)

**功能特性：**
- 账号CRUD操作
- **5步配置向导** (新功能)
- 批量操作
- 状态查看

**演示步骤：**
1. 点击"账号管理" → 进入账号列表
2. 点击"添加账号" → 启动5步配置向导
3. 完成向导：基本信息 → VPN配置 → 反检测 → 登录测试 → 会话管理
4. 查看创建的账号，可编辑、删除、测试

### 2. VPN配置管理 (VPNPage)

**功能特性：**
- VPN节点CRUD
- **账号-VPN关联管理** (新功能)
- 连接质量监控
- IP轮换策略

**演示步骤：**
1. 点击"VPN配置" → 查看VPN列表
2. 添加VPN节点 (OpenVPN/WireGuard)
3. 切换到"账号-VPN关联"标签
4. 为账号分配VPN，查看关联关系
5. 查看连接监控数据

### 3. 反检测配置管理 (AntiDetectionPage) - **全新功能**

**功能特性：**
- 浏览器指纹配置
- 人类行为模拟
- 设备伪装设置
- 预设模板库
- 批量应用到账号

**演示步骤：**
1. 点击"反检测配置" → 进入管理界面
2. 查看预设模板 (Chrome桌面、Safari移动端等)
3. 创建自定义配置
4. 测试配置有效性
5. 批量应用到多个账号

### 4. 登录状态监控 (LoginStatusPage) - **全新功能**

**功能特性：**
- 实时状态显示
- 健康状态仪表板
- 失败原因分析
- 自动重连配置
- 历史趋势图表

**演示步骤：**
1. 点击"登录状态监控" → 进入监控面板
2. 查看所有账号的实时状态
3. 点击单个账号查看详情
4. 查看健康状态仪表板
5. 配置自动重连策略

### 5. 任务调度管理 (TasksPage)

**功能特性：**
- 聊天任务调度
- 50个预设对话脚本
- 智能时间安排
- 执行监控

**演示步骤：**
1. 点击"任务调度" → 进入任务列表
2. 创建新任务，选择对话脚本
3. 配置执行时间和账号
4. 查看任务执行状态

## 📝 使用建议

### 最佳实践
1. **新账号配置**: 始终使用5步向导进行完整配置
2. **VPN管理**: 为每个账号分配独立的VPN节点
3. **反检测配置**: 定期更新配置，避免长期使用相同指纹
4. **状态监控**: 每天检查监控面板，及时处理问题账号
5. **任务调度**: 合理安排任务时间，避免高峰期操作

### 故障排除
1. **登录失败**: 检查VPN连接、反检测配置、账号状态
2. **验证码问题**: 调整反检测配置或更换IP
3. **会话过期**: 重新登录并保存新会话
4. **性能问题**: 检查VPN延迟和带宽

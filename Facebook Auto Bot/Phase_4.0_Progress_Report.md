## 技术实现详情

### 前端架构
- **React + TypeScript**：严格类型检查，提高代码质量
- **状态管理**：Zustand + React Query
  - Zustand用于客户端状态（实时数据）
  - React Query用于服务器状态（API数据）
- **UI组件库**：Ant Design v5
- **图表库**：Recharts
- **路由**：React Router DOM v6
- **PWA**：Vite PWA插件

### 核心组件
1. **DashboardPage**：主仪表板，集成所有核心功能
2. **RealTimeMonitor**：实时监控面板
3. **DashboardChart**：图表组件
4. **AccountBatchOperations**：批量操作组件
5. **TaskCreatePage**：任务创建向导
6. **TasksPage**：任务管理页面
7. **AccountDetailPage**：账号详情页面
8. **VpnConfigPage**：VPN/IP配置页面

### 实时数据流
```
WebSocket Server → useWebSocket → useDashboardWebSocket → dashboardStore → 组件更新
```
